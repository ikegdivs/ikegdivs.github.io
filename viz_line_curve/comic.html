D3 line styling: curve

When drawing a line, you can use the 'curve' method to modify the appearance of the line. The curve method causes the line connecting points to be drawn using curves or bends. This can be helpful when attempting to smooth lines or show transitions between categories. The curve method can use a function to change how this curve is drawn. This example extends the example located here.

Utilizing the user's selected curve type.

In order to show how the curve method can be implemented, you can add the following change to the selector's 'change' function. This code allows a user to change the appearance of a D3 line chart. 


// If the user changes the X slider, adjust the color of the chart line.
$('#formControlRangeX').on('input', function(event){
    // Get the input from the slider.
    userVal = parseFloat($(event.currentTarget).prop('value'));
    
    // Modify the user input to act as a color value
    colorVal = Math.round(userVal * 2.5);

    // Adjust the color of the chart line.
    body.selectAll('path')
        .attr('stroke', 'rgba('+ (255-colorVal) + ',' + colorVal + ', 0)')
})
